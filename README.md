## Run Locally

Clone the project

```bash
  git clone git@github.com:khleomix/khleomix.git
```

Go to the project directory

```bash
  cd khleomix
```

Install dependencies

```bash
  npm install --legacy-peer-deps
```

Start the server

```bash
  npm run start
```

Available Scripts

```bash
  npm run dev
  npm run build
  npm run lint
```
